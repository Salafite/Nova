# AdminView — API Endpoints

All endpoints are prefixed with `/api/T0021I`.

| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/T0021I/` | List all system users |
| POST | `/api/T0021I/` | Create a new user |
| PUT | `/api/T0021I/{id}` | Update an existing user |
| DELETE | `/api/T0021I/{id}` | Delete a user |
