# Payment Terms

**Program code:** `T0096` — Payment Terms

## Purpose
Manage payment terms used across sales and purchase transactions, including due days, early payment discounts, and default term flags.

## How to access
1. Log in to Nova ERP
2. Navigate to **Accounting & Finance > Payment Terms** in the sidebar
3. The screen loads at `#/payment-terms`

## Backing table
`Nova.T0096` — stores all payment term records.

## API prefix
`/api/T0096I`

## Permissions
- **Full access**: Admin, Manager
- **Read only**: Viewer

## Dependencies
- Depends on: none
- Used by: Sales Orders, Purchase Orders, Invoices
