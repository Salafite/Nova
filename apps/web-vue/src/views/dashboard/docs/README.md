# Dashboard View

- **Route:** `#/dashboard`
- **Component:** `DashboardView.vue`
- **Backend:** Aggregates from multiple T* endpoints (no dedicated dashboard API)
- **Purpose:** Landing page showing key business metrics and recent activity

## Features

- 8 stat cards with live counts from Products, Customers, Suppliers, Sales Orders, Invoices, Payments, Employees, Users
- Recent activity section showing latest 5 invoices
- Loading, error, and empty states
- Full RTL/i18n support
