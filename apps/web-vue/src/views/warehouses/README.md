# Warehouses

**Program code:** `T0008` (Warehouses)

## Purpose
Manage warehouse locations and storage facilities. Each warehouse is a physical or logical location where inventory is stored.

## How to access
1. Log in to Nova ERP
2. Navigate to **Foundation > Warehouses** in the sidebar
3. The screen loads at `#/warehouses`

## Backing table
`Nova.T0008` — stores warehouse records.

## API prefix
`/api/T0008I`

## Permissions
- **Full access**: Admin, Manager (can create, edit, delete)

## Dependencies
- Referenced by: T0009 (Stock Levels), T0064 (Stock Movements)
