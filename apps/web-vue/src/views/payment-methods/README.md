# Payment Methods

**Program code:** `T0097` — Payment Methods

## Purpose
Manage payment methods used across the system, such as cash, bank transfer, credit card, and cheque.

## How to access
1. Log in to Nova ERP
2. Navigate to **Accounting & Finance > Payment Methods** in the sidebar
3. The screen loads at `#/payment-methods`

## Backing table
`Nova.T0097` — stores all payment method records.

## API prefix
`/api/T0097I`

## Permissions
- **Full access**: Admin, Manager
- **Read only**: Viewer

## Dependencies
- Depends on: none
- Used by: Payments, Invoices
