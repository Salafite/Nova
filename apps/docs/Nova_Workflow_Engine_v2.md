# Nova Workflow Engine v2

## Engine

Event-driven approval engine.

## Components

-   Workflow Definitions
-   States
-   Transitions
-   Conditions
-   Actions
-   Notifications
-   SLA Timers
-   Escalations

## Sample Flow

Purchase Request -\> Manager Approval -\> Finance Approval -\> Purchase
Order -\> Goods Receipt -\> Invoice -\> Payment

## Supported Modules

Sales, Purchasing, HR, Manufacturing, Projects, Finance, Maintenance.
