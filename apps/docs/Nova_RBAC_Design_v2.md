# Nova RBAC Design v2

## Security Model

User -\> Role -\> Permission

## Scope

-   Tenant
-   Company
-   Branch
-   Department
-   Record

## Permission Types

Create, Read, Update, Delete, Approve, Export, Print, Configure.

## Roles

Super Admin, System Admin, Finance, Sales, Purchasing, Warehouse, HR,
Manufacturing, Manager, Employee, Auditor.
